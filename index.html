Windows Performance Tools

Quick access to important Windows settings for optimizing your PC.

  • Advanced System Settings: Press Windows + R, type SystemPropertiesAdvanced, press Enter.
  • Performance Options: Press Windows + R, type SystemPropertiesPerformance, press Enter.
  • Task Manager: Press Ctrl + Shift + Esc.
  • Device Manager: Press Windows + X and select Device Manager.
